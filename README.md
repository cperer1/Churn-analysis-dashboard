# Análisis de Cancelación de Clientes (Churn Analysis)

## Descripción del proyecto

Este proyecto analiza el comportamiento de cancelación de clientes en una empresa de telecomunicaciones basada en suscripciones mensuales.

El objetivo es identificar los principales factores que influyen en el churn y proponer estrategias para mejorar la retención de clientes.

---

## Tecnologías utilizadas

- PostgreSQL
- SQL
- Power BI
- DBeaver

## Principales hallazgos

- Los contratos **Month-to-Month presentan el mayor churn**
- El **abandono ocurre principalmente en los primeros 6 meses**
- Los clientes con **cargos mensuales altos cancelan más**
- El servicio de **Fiber Optic tiene mayor tasa de cancelación**
- Los clientes que usan **Electronic Check presentan mayor churn**

---

## Conclusiones

El churn está principalmente asociado con:

- Bajo compromiso contractual
- Alta sensibilidad al precio
- Experiencia inicial del cliente

---

## Recomendaciones

- Incentivar contratos a largo plazo
- Mejorar el onboarding de clientes
- Optimizar la experiencia del usuario
- Implementar estrategias de fidelización temprana
